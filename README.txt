 
				AMERICA VIRTUAL - FORECAST WEATHER



FORECAST WEATHER es una app web que informara sobre el pronostico del tiempo de distintas ciudades del mundo.
Una vez registrado en el sitema (mail y contrase√±a) se podra acceder a todas las funciones de la app.
Seleccionando un pais, y luego una ciudad, conseguira la informacion correspondiente a la zona indicada.


ANTES DE INICIAR - 

El sistema esta codificado en C# .NET MVC 5, conectado a una Base de Datos MySql. 

Ejecutar el script de la Base de Datos incluido en la ruta (~/Scripts/DBScript.sql).


DOCUMENTACION -

Las llamadas son dirigidas a la API Free de OpenWeatherMap  (https://openweathermap.org/)
Dirigiendose a la ruta del servidor, el sistema guiara el funcionamiento del mismo.
